# 📊 Blinkit Sales Dashboard Insights

## Overview

This project analyzes Blinkit sales data using Power BI to identify sales trends, outlet performance, and product-level insights.

---

## Key Insights

- Analyzed overall sales performance using KPI cards.
- Studied outlet-wise sales contribution.
- Compared sales based on outlet size and location.
- Analyzed product categories and item types.
- Evaluated customer ratings and average sales.
